## About This Repository

This repository contains all tasks completed as part of the CodeAlpha Web Development internship. Each task is a standalone full stack project built with Node.js, Express, MongoDB, and a vanilla HTML/CSS/JS frontend. Projects are structured as independent folders within this repo.

---

## Tasks

### Task 01 — DevCircle (Social Media Platform)
**Folder:** `CodeAlpha_DevCircle/`

A dark-themed developer social media platform with full auth, posts, comments, likes, follow system, notifications, user profiles with custom links, and @ mention support.

**Key Features:**
- JWT authentication (register / login)
- Create posts with image upload via Cloudinary
- Like, comment, delete posts and comments
- Follow / Unfollow users with suggestions sidebar
- User profiles with avatar, bio, and custom links
- @ mention autocomplete in posts and comments
- Notification system (likes, comments, follows, mentions) with 14-day TTL
- User search in navbar
- Seed data with 10 real dev community profiles

**Tech Stack:** Node.js, Express, MongoDB, Mongoose, JWT, Cloudinary, Multer, bcryptjs, HTML/CSS/JS

**Status:** `Completed ✓`
